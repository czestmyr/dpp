Option/value is optional. If not presented, default value is used.

bool options::lexical_cast ( std::string  value)

Specialized functions for casting text to boolean.

std::stringstream can't handle casting "false" to false or "true" to true values. We need to overload/specialize lexical_cast functions.

template<typename T >
T options::option_cast ( const option &  operand)

Casting of option value.

Option value is hidden behind placeholder that only has access to memory location of stored value. Cast is used to represent this memory location as memory location of specific type.

Parameters:
operandoption object which value we want to gain.
Returns:
Value of desired type stored in option.
